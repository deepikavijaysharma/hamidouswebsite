 Our Services

Deliver a Cloud Day

Category : Demand Generation

Description

Work with a customer to develop a vision for implementing Cloud technology in order to solve business problems with innovative solutions. Note: This service is typically provided using either “Cloud Accelerate for LOB” or “ECAL Cloud Vision” engagement frameworks. In either case, priority use cases are identified along with a conceptual/notional solution architecture. Storyboards, demos and/or prototypes may be used to prove out the Cloud Solution vision.

SE Role Engagement

ECAs and/or Industry Advisors typically lead this engagement and function as the “Ideators” in Cloud Accelerate engagements. Platform Specialists and/or Hub SEs participate as the “Prototypers” in any development of proofs of value.
