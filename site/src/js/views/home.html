Develop a Cloud Vision

Category : Demand Generation

Description

This Solution-Engineering-as-a-Service includes:
  • Working with a customer to develop a vision for implementing Cloud technology in order to solve business problems with innovative solutions.
  • Leveraging Cloud Accelerate for LOB” or “ECAL Cloud Vision” engagement frameworks.
  • Identifying priority use cases, along with a conceptual/notional solution architecture. storyboards, demos and/or prototypes to prove out the Cloud Solution vision.

SE Role Engagement

  • ECAs and/or Industry Advisors typically lead this engagement and function as the “Ideators” in Cloud Accelerate engagements.
  • Platform Specialists and/or Hub SEs participate as the “Prototypers” in any development of proofs of value.

Design a Cloud Solution

Category : Opportunity Execution

Description

This Solution-Engineering-as-a-Service includes:
  • Collaboratively working with a customer to turn their Cloud Vision into a robust Solution Presentation and Oracle Sales Proposal.
  • Producing a sales proposal, including developing logical and physical architectures, sizing current and future workloads, recommending phased implementation roadmaps, creating the intake provisioning document, solution presentation, and contributing to the Oracle Sales Proposal.
  • This service typically uses ECAL Stages 2-4 as the underlying execution framework.

SE Role Engagement

  • ECAs lead this engagement and are responsible for the majority of the deliverables.
  • Platform Specialists and Hub SEs work on Portfolio and Workload Analysis efforts, contribute to the physical architecture designs, and provide foundation services as needed (e.g. Customer Discovery of current state IT Portfolio, Customer Demos, Sizing/Quoting/Config).
  • This service may spawn the “Build a POC” service
  • Business Value Consultants support any value-based activities and deliverables, which can also fall under the “Build a Business Case” service.
  • CSMs are responsible for ECAL Migration and Adoption Planning activities and deliverables, and the subsequent “Adopt a Cloud Solution” service.
  • For more information, see our ECAL website
